# AdventureWorks Cycles Executive Dashboard

## Project Overview
This project is an interactive executive dashboard built in **Power BI** for AdventureWorks Cycles, a global bicycle retailer. The goal is to povide leadership with a single source of truth to monitor business performance, identify product trends, and uncover growth opportuities.

## Key Features
- **Executive KPIs**: Total Revenue, Total Orders, AOV, and Customer Count.
- **Product Performance**: Identifies top products and revenue concentration.
- **Customer Insights**: Analyzes repeat purchase behavior and retention.
- **Geographic & Seasonal Trens**: Pinpoints regional underperformance and sales seasonality.
- **Actionable Recommendations**: Data-driven strategies for business growth.

## Key Insights
- **Top 3 products** generates 86.2% of total revenue, indicating high concentration risk.
- **Repeat customers (39% of total)** drive 71.4% of revenue, highlighting a major retention opportunity.
- **Germany** signifcantly underperforms ($4.95M vs. $24.3M in the top territory).
- **Sales peak** in March, May, and October, with February being consistently weak.

## Tools Used
- **Power BI Desktop**: For data modelling and visualization.
- **DAX**: For creating measures like Total Revenue, Repeat Rate, etc.
- **Power Query**: For data transformation.

## How to Use
1. Clone this repository
2. Open the '.pbix' file in Power BI Desktop.
3. Interact with the dashboard using the slicer for year, territory, and product category.

## Files in This Repository
- 'AdventureWorks_Dashboard.pbix': The main Power BI  project file.
- '/data': Contains the dataset used for the analysis.
- '/screenshots': Preview images of the dashboard.
- 'Insights_Report.pdf': A detailed report on the analysis and recommendations.
